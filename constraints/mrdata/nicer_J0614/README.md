## Posterior samples

The posterior mass-radius MultiNest samples and associated contours for the headline ST+PDT inference run can be found in ``MR_samples_and_contours.tar.gz``. There, the 68%, 95% and 99% credible intervals mass-radius contours are provided respectively in ``J0614_STPDT_68.txt``, ``J0614_STPDT_95.txt``, and ``J0614_STPDT_99.txt`` files. The MultiNest equally weighted mass-radius posterior samples are in the ``J0614_ST_PDT_20kLP_0p05SE_0p1ET_mrsamples_post_equal_weights.dat`` file.
